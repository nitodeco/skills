# Skills

A collection of usefull AI agent skills that I wrote and use for my projects. For more information about skills, visit [agentskills.io](https://agentskills.io).

## Installation

The recommended way to install skills is using [skills.sh](https://skills.sh):

```bash
npx skills add nitodeco/skills
```

To update skills, use:

```bash
npx skills update nitodeco/skills
```

Alternatively, you can clone this repo and either copy the skills into your own skills directory or add symlinks to the skills in this repo.
